## Projects

### Basketball Analytics System
A complete statistical system built from scratch for a real recreational basketball group. Every stat manually tracked across 66 games over 9 months. Covers box scores, advanced metrics (eFG%, PIE, ORB%), duo chemistry, MVP modelling across 11 models, NBA archetype matching, and 2K-style player ratings across 10 interactive dashboards.

**Stack:** Vanilla JS, Chart.js, Python, pandas, NumPy, Excel/Google Sheets

### Hoopstats App *(in progress)*
The product version of the basketball project. A web app for any player to track game stats, store them online, and see automatic dashboards. Basketball today — baseball, soccer, and football coming next.

### Fraud Detection — E-Commerce
End-to-end ML pipeline on 1.47M transactions. Feature engineering, class imbalance handling, and a four-model comparison (Logistic Regression, Naive Bayes, Decision Tree, Random Forest).

**Stack:** Python, scikit-learn, pandas, Jupyter

### News-Driven Stock Movement Model
NLP pipeline using TF-IDF vectorization of financial news headlines to predict intraday equity returns. Strict temporal alignment to avoid look-ahead bias. 57% directional accuracy on held-out 2024 test data.

**Stack:** Python, TF-IDF, Ridge Regression, pandas, sklearn Pipeline
